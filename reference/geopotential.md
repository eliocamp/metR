# Geopotential height

Monthly geopotential field at 700hPa south of 20Â°S from January 1990 to
December 2000.

## Usage

``` r
geopotential
```

## Format

A data.table with 53224 rows and 5 variables.

- lon:

  longitude in degrees

- lat:

  latitude in degrees

- lev:

  level in hPa

- gh:

  geopotential height in meters

- date:

  date
